Design

Motivation

We want to define composable APIs that enable the creation of complex training loops and higher-level JAX frameworks.

total_steps = 10_000

_, history, state = loop(
    state,
    dataset,
    {
        every(steps=1): [
            train_step, 
            keras_bar(total=total_steps),
        ],
    },
    stop=total_steps,
)

Callback Spec

def callback(
    state, batch, [broadcasts, statics]
) -> (logs, state) | None:
Description Where: * `state` is a pytree containing the training state * `batch` is a pytree containing the batch data * `broadcasts` (optional) is a pytree containing additional inputs that can be broadcasted to all devices * `statics` (optional) is a hashable object containing additional static inputs `broadcasts` and `statics` are passed as positional arguments. Callbacks can return: * `logs` (optional) is a dict containing the log information * `state` (optional) is a pytree containing the updated training state Returns can be either a 2-tuple of `(logs, state)` or `None`, additionally `logs` or `state` can be `None` if they are not needed.


Loops

Loop callbacks

def callback(
    state, batch, [elapsed, loop]
) -> (logs, state) | None:
Description Loop callbacks implement the callback spec with the following instantiation: * `elapsed` (optional) is a pytree containing information about steps, samples, time, and date since the start of the loop. * `loop` (optional) is a regular python object that runs the loop.


Schedules

def schedule(elapsed: Elapsed) -> bool:
Description `schedule`s take in an `elapsed` instance and returns a boolean indicating whether that time is within the schedule.


Loops

def loop(
    state, dataset: [batch], tasks: {schedule: [callback]}
) -> (state, [logs], elapsed):
Description Loops are comprised of a `state` that is threaded through the loop, a `dataset` that is iterated over, and a dictionary of `schedules` to `callbacks` that execute various tasks such as training, logging, evaluation, etc. Loops return the final `state`, a list of `logs` from each step, and an `elapsed` instance.


Examples

Sample Usage with Flax 
@jax.jit
def train_step(state: TrainState, batch):
    inputs, labels = batch["image"], batch["label"]

    def loss_fn(params):
        logits = state.apply_fn({"params": params}, inputs)
        loss = optax.softmax_cross_entropy_with_integer_labels(
            logits=logits, labels=labels
        ).mean()
        return loss, logits

    grad_fn = jax.value_and_grad(loss_fn, has_aux=True)
    (loss, logits), grads = (state.params)
    state = state.apply_gradients(grads=grads)
    logs = {
        "loss": loss, 
        "accuracy": jnp.mean(jnp.argmax(logits, -1) == labels)
    }
    return logs, state


_, history, state = loop(

        ],
        every(100): [
            Profile(logdir=logdir), # clu.periodic_actions
        ],
    },
    stop=total_steps,
)
